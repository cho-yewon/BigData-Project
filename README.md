# 🐾 전국 반려동물 등록·입양·유기 패턴 분석 및 예측

## 📊 분석할 내용

* 전국 **반려동물 등록 현황**, **유기·입양 동물 발생 추이**, **보호소 현황**의 시계열·공간적 패턴 분석
* 동물종(개/고양이), 연령, 성별, 중성화 여부, 보호소 위치 등에 따른 **유기·입양률 차이** 분석
* 계절, 요일, 지역(시·도 단위) 등의 변수와 **유기·입양 건수의 상관관계** 파악
* 머신러닝/통계모델을 활용한 **향후 유기동물 발생·입양률 예측 모델** 구축

## 📌 프로젝트 범위

### 1️⃣ 데이터 수집 / 크롤링

* [국가동물보호정보시스템 Open API](https://www.data.go.kr/data/15098931/openapi.do)를 이용해 **전국 유기·입양 동물 데이터 수집**
* [전국동물보호센터 정보표준데이터 Open API](https://www.data.go.kr/data/15025454/standard.do)를 통해 **보호소 위치·운영 현황 데이터 수집**
* [농림축산식품부 농림축산검역본부_동물등록 현황 Open API](https://www.data.go.kr/data/15103156/openapi.do)를 이용해 **전국 시·도별 등록 데이터 자동 수집**

### 2️⃣ 데이터 저장 / 추출

* 수집한 모든 데이터를 **MongoDB**에 통합 저장
* 데이터셋(유기·입양 / 보호소 / 등록현황)을 **각각의 컬렉션(collection)** 으로 관리
* Open API의 JSON 응답을 변환 없이 MongoDB에 저장하여 효율적 관리

### 3️⃣ 데이터 가공 / 정제

* Pandas로 결측치 및 이상치 처리
* 날짜, 지역, 동물종, 보호소, 성별, 중성화 여부 등 주요 변수 정제
* 계절·요일·지역코드 등 **파생 변수(Feature Engineering)** 추가 생성

### 4️⃣ 데이터 분석

* Python + scikit-learn으로 유기·입양 패턴 분석 및 예측 모델 구축
* **시계열 분석(Time Series)**, **지역별 통계 분석(Spatial Analysis)** 수행
* 동물종·연령·성별 등 다변수 상관관계 도출

### 5️⃣ 데이터 시각화

* Matplotlib, Seaborn, Tableau를 활용한 시각화
* **지도 기반 핫스팟 분석(지역별 유기·입양 현황)**
* **시계열 그래프(월별·계절별 발생 추이)** 및 **Heatmap(지역-시점별 패턴)** 제작

### 6️⃣ 분석 결과

* 전국 단위의 유기·입양 패턴 및 예측 결과 도출
* 지역별 특징·문제점을 도표 및 지도 형태로 시각화
* 정책 개선·보호소 자원 배분·입양 캠페인 기획 등 **정책적 시사점 제시**

## 🤖 생성형 AI 결과

**입력**
주제 ‘전국 반려동물 등록·입양·유기 패턴 분석 및 예측’에 대해, 이 주제가 과제·프로젝트용으로 적합한지와 사회적·정책적·기술적 가치, 아이디어·분석·시각화 방법, 강점·한계·발전방향을 모두 포함해 보고서에 바로 넣을 수 있는 ‘생성형 AI 결과’ 항목(5~6문장)으로 작성해줘.

**결과**
본 주제는 전국 단위의 반려동물 등록·입양·유기 데이터를 활용하여 유기동물 발생 추이와 입양률을 분석·예측하는 연구로, 공공데이터(Open API)를 적극적으로 활용할 수 있어 과제·프로젝트용으로 매우 적합하다.
사회적으로는 유기동물 문제 완화와 입양 문화 확산, 정책적으로는 지역별 보호소 운영 효율화 및 반려동물 정책 개선에 기여할 수 있다.
기술적으로는 시계열 분석, 지도 기반 핫스팟 시각화, 머신러닝 예측모델(XGBoost, RandomForest 등)을 통해 과학적 근거를 제시할 수 있다.
강점은 데이터의 공개성과 전국 단위의 폭넓은 분석 가능성에 있으며, 한계는 민간 보호소 데이터나 개체별 세부 정보 부족으로 인한 세밀한 분석 한계가 존재한다.
이를 보완하기 위해 추가적인 지역 데이터 연계 및 변수 확장을 통한 예측 정확도 향상이 필요하다.
종합적으로 본 프로젝트는 공공데이터 기반 AI 분석을 통해 사회문제 해결과 정책혁신을 동시에 달성할 수 있는 실용적 빅데이터 과제로 평가된다.
